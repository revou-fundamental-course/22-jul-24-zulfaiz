Mendesain website

Saya memiliki keahlian dalam mendesain dan mengembangkan website yang menarik dan responsif. Saya telah mengerjakan beberapa proyek desain web untuk berbagai klien dan memastikan setiap website yang saya buat memiliki tampilan yang menarik, fungsionalitas yang baik, dan pengalaman pengguna yang optimal. Saya menggunakan kombinasi HTML, CSS, dan JavaScript untuk menciptakan desain yang estetis dan interaktif. Saya juga memperhatikan kebutuhan dan preferensi klien untuk menghasilkan solusi desain yang sesuai dengan tujuan bisnis mereka.

Gaming

Saya telah mengembangkan berbagai permainan yang menarik dan inovatif. Proyek-proyek saya mencakup pengembangan game PC, game mobile, dan game konsol. Saya telah bekerja dengan tim yang berdedikasi untuk menciptakan pengalaman bermain yang menghibur dan menarik bagi para pemain. Robotic

Proyek robotic ini merupakan hasil dari penggunaan platform Arduino dan komponen elektronik lainnya. Saya merancang dan memprogram robot ini untuk dapat melakukan tugas-tugas tertentu, seperti mengikuti garis, menghindari rintangan, dan lain sebagainya. Proyek ini melibatkan pemrograman bahasa C++ dan penggunaan sensor-sensor untuk mengambil input dari sekitar robot. Saya sangat menikmati proses merakit dan menguji robot ini, serta mengembangkan kemampuannya secara bertahap.
